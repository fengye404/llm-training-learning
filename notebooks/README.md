# Notebook 学习入口

你可以在 GitHub 里直接打开这些 `.ipynb` 预览内容，也可以本地用 Jupyter 运行。

## 本地运行方式

```bash
pip install notebook
jupyter notebook
```

然后在浏览器中打开对应 `.ipynb`。
