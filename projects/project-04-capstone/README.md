# 第 5 章代码：成果沉淀

## 这个脚本教你什么

- 如何把实验结果汇总成可分享的 Markdown 报告。
- 如何把学习过程沉淀成工程化产物。

## 运行方式

```bash
python3 projects/project-04-capstone/build_learning_report.py
```

输出文件：

- `projects/project-04-capstone/learning_report.md`

## Java 对照理解

- `Experiment` dataclass 可类比 Java DTO。
- `to_markdown(...)` 可类比报告组装服务。
