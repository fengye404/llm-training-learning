# Phase 5 Code: Capstone Output

## What this script teaches

- How to summarize experiments into a shareable markdown report.
- How to turn learning into reusable engineering output.

## Run

```bash
python3 projects/project-04-capstone/build_learning_report.py
```

Output file:

- `projects/project-04-capstone/learning_report.md`

## Java mapping

- `Experiment` dataclass ~= a Java DTO.
- `to_markdown(...)` ~= report assembler service.
